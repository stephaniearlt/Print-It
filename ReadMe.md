# Print-It WebSite v.1

Lucien, directeur de l'imprimerie familiale "Print it", souhaite dynamiser le site Internet statique de l'imprimerie en ajoutant un carrousel interactif pour mettre en valeur leurs produits et services. 

À la fin du projet :

- Mis en place d'un carrousel interactif fonctionnel, ergonomique, intuitif et esthétiquement agréable
- Correction des bugs 
- Implémentation de la boucle infinie pour assurer un fonctionnement fluide du carrousel
